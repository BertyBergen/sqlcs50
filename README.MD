# Тесты для sqlcs50

## Как запустить тесты:

1. Установи `pytest`, если ещё нет:

   ```
   pip install pytest
   ```

2. Перейди в папку с проектом.

3. Запусти все тесты:

   ```
   pytest
   ```

4. Или запусти только один тестовый файл:

   ```
   pytest tests/test_insert_select.py
   ```

5. Или запусти тесты с фильтром по имени теста:
   ```
   pytest -k "insert"
   ```
